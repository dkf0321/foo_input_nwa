# Introduction

This is a foobar2000 component for playing NWA files.  

# Build from Source

This project is developed based on the foobar2000 SDK (version: 2025-03-07).  

Please download the SDK from the official source first and make sure to name the SDK folder SDK-2025-03-07, and place it in the same parent directory as the project directory.
